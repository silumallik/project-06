## Family Golf Website – Recreated UI

A modern and interactive **Family Golf website recreation** built to practice advanced **frontend animations, smooth scrolling, and responsive UI design**.
This project focuses on delivering a visually engaging user experience using smooth transitions, animated sections, and slider components inspired by real-world golf club websites.

The website showcases elegant layouts, animated text reveals, smooth scrolling effects, and interactive sliders, making it ideal for learning **animation-driven UI development**.

## Features

* Smooth scrolling experience
* Scroll-based animations and reveals
* Modern UI with responsive design
* Interactive sliders and carousels
* Clean and minimal layout
* Performance-optimized animations

## Technologies Used

* **HTML5** – Structure and semantic markup
* **CSS3** – Styling, layout, and responsiveness
* **JavaScript (ES6)** – Interactivity and animations
* **GSAP** – High-performance animations
* **Swiper.js** – Touch-enabled sliders and carousels

## Learning Outcome

* Implementing smooth scrolling with Locomotive Scroll
* Creating scroll-triggered animations using GSAP
* Integrating Swiper.js for sliders
* Building animation-rich websites
* Structuring clean and maintainable frontend code
